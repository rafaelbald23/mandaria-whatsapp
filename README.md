# MandarIA

Sistema de Disparo Automatizado de Mensagens WhatsApp

Desenvolvido por **MonitorIA**

## Descrição

MandarIA é um sistema web completo para envio automatizado de mensagens em massa via WhatsApp Web. Desenvolvido com Flask, Socket.IO e Selenium.

## Funcionalidades

- Envio de mensagens em massa
- Suporte a múltiplas mensagens por contato
- Envio de imagens
- Dashboard com estatísticas em tempo real
- Histórico de envios
- Relatórios detalhados
- Sistema de autenticação
- Gerenciamento de clientes (admin)

## Tecnologias

- Python 3.8+
- Flask
- Flask-SocketIO
- Selenium
- Pandas
- OpenPyXL

## Instalação

1. Clone o repositório
```bash
git clone <seu-repositorio>
cd web
```

2. Instale as dependências
```bash
pip install -r requirements.txt
```

3. Execute o servidor
```bash
python app.py
```

4. Acesse `http://localhost:5000`
